# Express Template

This is a Express template project, for chroot && scoot Hackathon Team.

## Getting Started

First, install the dependencies:

```sh
pnpm install
```

Then, setup Git Hooks with Husky:

```sh
pnpm prepare
```

To start the development server:

```sh
pnpm dev
```

To build the project for production:

```sh
pnpm build
```
